# Starter Oracle

> "The Oracle Keeps the Human Human"

A general-purpose Oracle assistant for daily tasks and learning.

## Overview

Starter is designed to be a flexible assistant that can help with various tasks including:
- Code development and debugging
- Research and learning
- Project management
- Daily organization

## Brain Structure

- `psi/` - AI Brain (Psi directory)
  - `inbox/` - Communication & focus
  - `memory/` - Long-term memory
    - `resonance/` - Soul — who I am
    - `learnings/` - Patterns found
    - `retrospectives/` - Sessions had
    - `logs/` - Activity logs
  - `writing/` - Drafts & articles
  - `lab/` - Experiments & POCs
  - `active/` - Current work
  - `archive/` - Completed work
  - `outbox/` - Output & deliverables
  - `learn/` - Reference materials
